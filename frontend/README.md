# MarketPulse AI Frontend

This folder contains all Streamlit frontend applications.

## Files:
- `app.py` - Main demo application (for deployment)
- `streamlit_cloud_app.py` - Cloud-optimized demo version
- `streamlit_production_app.py` - Production version (connects to API)

## Usage:
- For local demo: `streamlit run app.py`
- For cloud deployment: Use `app.py` as main file
- For production: Use `streamlit_production_app.py` with API backend
